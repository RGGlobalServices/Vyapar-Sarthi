# Vyapar Sarthi — User Manual

A guide for shopkeepers using Vyapar Sarthi to run their store: billing,
stock, udhar, staff and reports.

This manual describes the app as a shop owner sees it. It does not cover
installation or development — see the project `README.md` for that.

## A note on what you see

Two settings change what appears in your app, so your screen may not match
every page of this manual:

- **Your package** (Dukaan, Vyapar or Udyog) decides which sections appear in
  the sidebar. See [Packages & Plans](packages.md).
- **Your business category** (Kirana, Pharmacy, Electronics and so on) changes
  some labels and fields — for example, Electronics shops get EMI options and
  serial numbers, while Apparel shops get size variants.

Where a feature belongs to only one package, this manual says so.
