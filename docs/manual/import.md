# Import Data

Bringing your existing records into the app, instead of typing them in again.

Import Data is available on the **Vyapar** and **Udyog** packages.

## What you can import

| Import | Use it for |
|---|---|
| **Purchase Bills** | Bills from your suppliers |
| **Sales History** | Sales you recorded before using this app |
| **Bulk Stock** | A whole product list with quantities |
| **Udhar Khata** | Your existing credit ledger — who owes what |

## How it works

1. Open **Import Data**.
2. Choose what you're importing.
3. Follow the on-screen instructions for that type and upload your file.
4. Review what the app found before confirming.

## Before you import

**Check the preview carefully.** Importing writes real records — products,
stock, bills and balances — and undoing a bad import means cleaning up by hand.
It's worth reading the review step properly rather than clicking through.

Start with a small file to see how your data comes through, then import the
rest once you're happy.

**Import your udhar khata early.** Once customers exist with the right opening
balances, every later repayment lands in the right place.

## If you're on Dukaan

Import Data isn't part of the Dukaan package. You can enter products and udhar
customers by hand, or upgrade to Vyapar — see [Packages & Plans](packages.md).
