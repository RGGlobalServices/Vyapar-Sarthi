# रेफर करें और कमाएं

दूसरे दुकानदारों को बुलाएँ और इनाम पाएँ।

यह सुविधा **व्यापार** और **उद्योग** पैकेज में मिलती है।

## यह कैसे काम करता है

1. **रेफर करें और कमाएं** खोलें, वहाँ **आपका कोड** मिलेगा।
2. उसे भेजें — **लिंक कॉपी करें** या **WhatsApp पर शेयर करें**।
3. वे आपके कोड से साइन अप करें।
4. उनकी सब्सक्रिप्शन पूरी होने पर आपको इनाम मिलता है।

## आपको क्या मिलता है

स्क्रीन पर दिखता है: आपके **कुल रेफरल**, कितने **सफल** रहे, और आपने कितने
**मुफ़्त महीने** कमाए।

## उन्हें क्या मिलता है

रेफरल कोड से साइन अप करने वाले को सामान्य 7 दिन की जगह **14 दिन का मुफ़्त ट्रायल**
मिलता है — यानी भुगतान का फ़ैसला करने से पहले ही आपका कोड इस्तेमाल करना उनके लिए
फ़ायदेमंद है।

## टीम

अगर आप कई दुकानदारों को रेफर करते हैं, तो वे **टीम** में दिखते हैं, ताकि आप देख
सकें कि आपके ज़रिए कौन-कौन जुड़ा।
